# Release Readiness

Prepare evidence for a safe release.

## Checklist

1. Confirm scope and release candidate.
2. Confirm tests and quality gates passed.
3. Identify migrations, feature flags, secrets, and configuration changes.
4. Define rollback steps.
5. Define monitoring and alert checks.
6. Identify customer/user impact.
7. Require explicit approval before production deployment or irreversible actions.
